## Projects

### 1. [Knowledge Base-Driven Q&A Platform (KBQA)](./kbqa-platform/)

A RAG-based personalized intelligent Q&A system with vector semantic search, knowledge graph visualization, and multi-turn dialogue.

- **Tech Stack:** Python Flask, ZhipuAI Embedding-3/GLM-4, scikit-learn, ECharts, GitHub Actions CI/CD, Render
- **Live Demo:** [https://kbqa-system.onrender.com](https://kbqa-system.onrender.com)
- **My Role:** Core architect — designed and implemented the entire backend, vector knowledge base, knowledge graph, testing suite, CI/CD pipeline, and cloud deployment

### 2. [Memory-Augmented VLM-Driven Tabletop Arrangement Robot](./vlm-robot/)

A simulation system where a Panda robotic arm understands open-vocabulary natural language commands via Qwen2.5-VL and performs multi-step pick-and-place operations in ManiSkill3.

- **Tech Stack:** Python, ManiSkill3, Qwen2.5-VL, Numerical Jacobian IK, PandaArm Motion Planning
- **My Role:** VLM interface core developer — integrated Qwen2.5-VL, built 3-level JSON parsing fallback, semantic name mapping, and prompt engineering

### 3. [Minesweeper Rogue (扫雷地牢)](./boom-game/)

A Roguelike minesweeper game with IQ damage system, chain explosions, talent/skill systems, and room-based dungeon progression — built with Python + Pygame.

- **Tech Stack:** Python, Pygame
- **My Role:** Sole developer — designed and implemented all game systems (core mechanic, IQ system, chain explosions, talents/skills, menu/settings/tutorial)
